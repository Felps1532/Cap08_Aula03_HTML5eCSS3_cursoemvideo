Principais formatações:

Negrito / Destaque:

Nesse texto temos um termo em negrito usando a tag <b>.(não semântico!)

Nessa frase temos um termo em destaque usando a tag <strong>.(SEMÂNTICO!)

Itálico / Ênfase:

Nessa frase temos um termo em Itálico, usando a tag <i> (não semântico!)

Nessa frase temos um termo em ênfase usando a tag <em>(SEMÂNTICO)

Texo marcado:

Podemos também criar um texto marcado, com a tag <mark>(SEMÂNTICO)

E nesse parágrafo temos outro texto marcado.

Texto grande e pequeno:

Estamos criando um texto grande e um texto pequeno nesse parágrafo;

Texto deletado:

Podemos marcar um texto como excluído para indicar que ele pode ser lido, mas não considerado. Tag <del>(SEMÂNTICO)

Texto inserido:

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Tag <ins>.(SEMÂNTICO)

existe também o sublinhado com a tag <u>.(não semântico)

Texto sobrescrito:

Para inserir coisas do tipo x20+3. Tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O. Tag <sub>.
